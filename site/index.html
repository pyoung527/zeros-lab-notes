박제로의 실험기록 / Zero’s Lab Notes

AI로 내 삶과 일을 자동화하는 실험을 시작합니다

박제로의 실험기록은 AI 에이전트, 자동화 시스템, 생산성 도구를 직접 만들고 적용해보는 실험 로그입니다. 지금은 초기 단계라, AI를 실제 업무/일상/사이드 프로젝트에 적용하고 싶은 분들의 문제를 듣고 함께 작은 실험을 설계하고 있습니다.

실험에 참여하기 어떤 실험인지 보기

유료 상품을 강하게 판매하기보다, 실제로 반복되는 문제를 먼저 듣고 있습니다.

Problem

AI 툴은 많은데, 내 일에 붙이는 방식은 아직 막막합니다

툴은 써봤지만 시스템은 없다

ChatGPT나 AI 툴은 써봤지만, 반복 가능한 업무 흐름으로 만들지는 못한 상태.

비슷한 정리 작업을 반복한다

문서, 메시지, 콘텐츠 아이디어, 메모, 할 일을 매번 손으로 다시 정리하는 문제.

실행 구조가 없다

사이드 프로젝트와 공부 계획은 있지만 다음 행동으로 이어지는 구조가 약한 상태.

프롬프트보다 워크플로우가 필요하다

단발성 답변이 아니라 내 생활과 일에 반복적으로 붙는 작은 자동화 시스템이 필요한 경우.

Experiment Framing

완성된 정답보다 작은 문제를 찾아 설계하고 검증합니다

박제로는 AI 에이전트와 자동화 시스템을 직접 만들고, 실제 생활과 업무에 적용하며 기록합니다.

완벽한 솔루션을 먼저 약속하기보다, 구체적인 반복 문제를 찾고 작은 자동화 실험으로 검증하는 방식을 선호합니다.

지금은 방문자의 문제를 듣고, 반복되는 패턴을 기반으로 베타 진단·템플릿·서비스 가능성을 확인하는 단계입니다.

Offer Now

현재 제안하는 세 가지

아직 검증된 고가 상품이 아닙니다. 문제를 구조화하고, 가능성이 높은 경우 작은 실험을 함께 설계합니다.

01

AI 자동화 문제 진단

반복 업무, 일상 관리, 콘텐츠, 공부, 사이드 프로젝트 문제를 구조화합니다.

02

베타 진단 세션

가능성이 높은 문제에 대해 1:1로 작은 자동화 실험 설계를 도와봅니다.

03

실험 참여자 모집

향후 콘텐츠, 템플릿, 서비스 실험에 참여할 관심자를 모읍니다.

자동화 결과나 수익을 보장하지 않습니다. 실제 문제와 현재 도구 수준에 맞춰 작은 실험부터 설계합니다.

For / Not For

이런 분께 맞습니다

For

  • 반복되는 업무·정리·콘텐츠·사이드 프로젝트 문제가 있다.
  • AI를 실제 생활이나 일에 적용하고 싶다.
  • 완성된 정답보다 함께 실험하고 개선하는 방식이 좋다.
  • 과장된 수익 주장보다 현실적인 자동화 설계를 원한다.

Not for

  • 하루 만에 큰 수익을 보장받고 싶다.
  • AI가 모든 일을 알아서 대신해주길 기대한다.
  • 본인의 문제 설명 없이 완성된 답만 원한다.
  • 검증 전 고가 상품을 즉시 구매하고 싶은 사람만 대상으로 하지는 않는다.
