## 2. Shot-by-shot checklist

### A-roll

- 오프닝 Hook: 정면, 상반신, 00:00-00:28.
- 문제 설명: 정면, 살짝 느린 톤, 00:28-00:55.
- Hina 정의: 정면, 안정적인 톤, 00:55-01:25.
- 사용 사례 3가지: 정면 + 손동작 살짝, 01:25-02:15.
- 결과/한계: 정면, 조금 더 솔직한 톤, 03:05-04:25.
- 비개발자 의미: 정면, 친근하게, 04:25-05:00.
- 다음 예고/CTA/클로징: 정면, 05:00-06:25.

### B-roll

- 책상 위 노트북 열려 있는 장면.
- 메모장/노션/할 일 목록을 스크롤하는 손.
- 커피 또는 물컵 옆에 노트와 펜.
- 키보드 타이핑 클로즈업.
- 화면을 보며 생각하는 옆모습.
- 체크리스트에 표시하는 장면.
- 여러 프로젝트 메모가 흩어진 느낌의 컷.

### Screen recordings

- Hina에게 요청하는 프롬프트 입력 화면.
- Hina가 응답 초안을 생성하는 화면.
- 생성된 초안을 Young이 수정하는 장면.
- 체크리스트/대본 문서가 정리되는 화면.
- 개인정보/토큰/채팅방 이름/민감 파일 경로는 반드시 블러 처리.

### Graphics

- 시리즈 로고: “박제로의 실험기록 / Zero’s Lab Notes”.
- 에피소드 타이틀: “EP.01 Hina: 내 AI 에이전트 실험”.
- 3가지 사용처 그래픽: 기억 보조 / 업무 정리 / 초안 만들기.
- 한계 그래픽: 그럴듯한 오류 / 맥락 누락 / 설계 필요.

## 7. Filming setup checklist

### Camera

- 렌즈 닦기.
- 4K 가능하면 4K, 아니면 1080p 30fps.
- 눈높이보다 살짝 위에 카메라.
- 얼굴은 프레임 중앙보다 살짝 좌/우 여백.
- 머리 위 공간 너무 많이 두지 않기.
- 오프닝/클로징은 2테이크 이상.

### Audio

- 가능하면 핀마이크 또는 USB 마이크 사용.
- 에어컨/선풍기/키보드 소음 끄기.
- 10초 테스트 녹음 후 확인.
- 파열음 방지: 마이크를 입 정면보다 살짝 옆.
- 화면 녹화 음성은 따로 쓰지 말고 A-roll 음성 위주.

### Light

- 얼굴 앞 45도 방향에 메인 조명.
- 모니터 빛만으로 촬영하지 않기.
- 배경이 너무 어두우면 작은 보조등 켜기.
- 안경 반사 있으면 조명 각도 위로 조정.

### Background

- 책상 위 70%만 정리. 너무 완벽할 필요 없음.
- AI 실험실 느낌: 노트북, 노트, 펜 정도.
- 빨래/개인정보/택배 송장/지저분한 케이블 제거.
- 배경에 브랜드/시리즈 느낌 줄 작은 오브젝트 있으면 좋음.

### Privacy

- 화면 녹화 전 알림 끄기.
- 브라우저 북마크바/이메일/채팅 목록 숨기기.
- API 키, 토큰, 파일 경로, 개인 대화, 고객 정보 블러.
- 계정명 노출 확인.
- 업로드 전 1.5배속으로 전체 영상 보며 개인정보 체크.

## 8. 60분 최소 촬영 플랜

### 목표

- 완벽한 영상이 아니라 “첫 실험기록을 오늘 찍고 업로드 가능한 재료 확보”.

### 0-10분: 세팅

- 카메라/마이크/조명 세팅.
- 테스트 녹음 10초.
- 화면 녹화용 창 정리.
- 민감 정보 숨기기.

### 10-30분: A-roll 촬영

- Hook 3테이크.
- 전체 대본은 문단별로 끊어서 촬영.
- 실수해도 멈추지 말고 문장 처음부터 다시 말하기.
- 반드시 확보할 문장:
  - “Hina는 제가 만든 개인용 AI 에이전트입니다.”
  - “완벽한 AI 비서는 아닙니다.”
  - “대신 시작 속도를 올려줍니다.”
  - “여러분이라면 어떤 일을 맡기고 싶으세요?”

### 30-42분: Screen recording

- Hina 프롬프트 입력 장면.
- 응답 생성 장면.
- 초안을 수정하는 장면.
- 민감 정보 없는 샘플 화면만 사용.

### 42-52분: B-roll

- 타이핑 손 2컷.
- 노트/체크리스트 2컷.
- 노트북 화면 보는 옆모습 1컷.
- 책상 와이드 1컷.

### 52-60분: 안전 컷

- 클로징 한 번 더 촬영.
- 썸네일용 표정 5장 캡처:
  - 진지한 표정
  - 살짝 놀란 표정
  - 노트북 가리키는 표정
  - 생각하는 표정
  - 미소 짓는 표정
- 파일 백업.

## 최종 제작 원칙

- “내가 답을 안다”가 아니라 “직접 실험해봤다”.
- “AI로 돈 벌었다”가 아니라 “반복 업무를 줄일 수 있는지 봤다”.
- “완벽한 비서”가 아니라 “아직 어설프지만 쓸모 있는 파트너”.
- 첫 영상이므로 신규 시청자에게 채널의 약속을 명확히 전달한다.
- 댓글/DM은 다음 실험 주제 검증 신호로 사용한다.
