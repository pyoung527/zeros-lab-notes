박제로의 실험기록 / Zero’s Lab Notes

AI는 써봤는데, 내 일에 어떻게 붙여야 할지 모르시나요?

박제로 / Young Park. AI 정규 학위 + CERN 협업 연구실에서 측정 자동화 실무. 지금부터 클라이언트 자동화 프리랜싱을 시작합니다. 그 과정을 공개 실험 로그로 함께 쌓아갑니다. 0번째 실험 진행 중. 첫 영상이 공개되면 알림으로 보내드려요.

실험 알림 받기 (무료, 1분) 자동화 상담 신청 (베타, 무료)

가벼운 트랙은 첫 영상·실험 노트 알림, 무거운 트랙은 30분 베타 진단입니다.

Problem

AI 툴은 많은데, 내 일에 붙이는 방식은 아직 막막합니다

툴은 써봤지만 시스템은 없다

ChatGPT나 AI 툴은 써봤지만, 반복 가능한 업무 흐름으로 만들지는 못한 상태.

비슷한 정리 작업을 반복한다

문서, 메시지, 콘텐츠 아이디어, 메모, 할 일을 매번 손으로 다시 정리하는 문제.

실행 구조가 없다

사이드 프로젝트와 공부 계획은 있지만 다음 행동으로 이어지는 구조가 약한 상태.

프롬프트보다 워크플로우가 필요하다

단발성 답변이 아니라 내 생활과 일에 반복적으로 붙는 작은 자동화 시스템이 필요한 경우.

Experiment Framing

완성된 정답보다 작은 문제를 찾아 설계하고 검증합니다

아직 검증된 자동화 전문가라고 말하지 않겠습니다. 근거 있는 시작 단계에서, 실제 문제를 작게 쪼개 공개 실험과 콘텐츠로 기록합니다.

완벽한 솔루션을 먼저 약속하기보다, 구체적인 반복 문제를 찾고 작은 자동화 실험으로 검증하는 방식을 선호합니다.

흐름은 단순합니다. 본업 자동화와 진단 세션에서 나온 실제 문제 → 일반화 가능한 패턴을 공개 실험으로 정리 → 유튜브·알림으로 공유 → 비슷한 문제를 가진 분들의 진단 의뢰로 다시 순환.

Trust Assets

왜 이 실험에 합류해도 되는지

저는 검증된 자동화 전문가가 아닙니다. 지금 시작하는 단계입니다. 다만 시작점은 갖추고 있습니다.

AI 정규 학위

캐나다 Centennial College AI Advanced Diploma (2021-2023)

측정 자동화 실무

경북대 고에너지물리연구센터 / CERN HL-LHC CMS MTD 협업. 실리콘 검출기(LGADs) QA/QC, 측정 자동화·데이터 분석.

B2B 기술영업 경력

Keysight Technologies에서 기업·연구기관 솔루션 매칭 경험. 고객 문제를 듣고 풀어내는 훈련.

해외 원문 도구를 빠르게 흡수

전자전기공학 학부 + 영어 C1. 최신 AI 도구·문서·논문을 바로 읽고 적용.

이 조합으로, 측정 자동화에서 일반 업무 자동화로 영역을 확장하고 있습니다.

Automation Patterns

지금까지 해온 자동화 (개인/연구 영역)

외부 클라이언트 사례가 아니라, 실제로 만들고 쓰는 자동화 패턴입니다.

연구실 측정 데이터 자동 정리·시각화 파이프라인

실험 데이터를 반복 가능한 형태로 정리하고, QA/QC 판단에 필요한 시각화로 이어가는 흐름.

개인 학습·아이디어 로그 정리 자동화

메모, 학습 기록, 콘텐츠 아이디어를 다시 찾고 다음 행동으로 옮기기 쉽게 정리하는 구조.

반복되는 영문 커뮤니케이션 템플릿화

자주 쓰는 영문 메시지와 설명을 상황별 템플릿으로 만들어 응답 시간을 줄이는 방식.

0번째 실험 진행 중

아직 결과를 과장하지 않습니다. #1부터 받아보고 싶은 분께 첫 공개 실험 알림을 보냅니다.

Diagnostic Session

베타 진단 세션 (현재 무료)

AI를 내 업무에 붙일 수 있을지 먼저 확인하는 30분 진단입니다.

  • 형식: 화상 30분 (Zoom/Google Meet) — 비동기 문서 가능
  • 비용: 무료 (베타 단계)
  • 인원: 주 2명 한정 (본업 병행 중이라 시간이 한정적입니다)
  • 조건: 진단 내용은 익명 처리 후 콘텐츠 소재로 활용될 수 있습니다 (사전 동의)
  • 이후: 직접 적용 가능한 문제 → 가이드·템플릿 제공 / 더 깊은 작업이 필요한 문제 → 프로젝트 견적 안내

이 세션은 본업 프리랜서 작업의 사전 검토 역할도 합니다. 가벼운 자문으로 끝날 수도, 본격적인 의뢰로 이어질 수도 있습니다. 정직하게 말씀드리면 — 양쪽 다 환영합니다.

For / Not For

이런 분께 맞습니다

For

  • 반복되는 업무·정리·콘텐츠·사이드 프로젝트 문제가 있다.
  • AI를 실제 생활이나 일에 적용하고 싶다.
  • 완성된 정답보다 함께 실험하고 개선하는 방식이 좋다.
  • 과장된 수익 주장보다 현실적인 자동화 설계를 원한다.

Not for

  • 하루 만에 큰 수익을 보장받고 싶다.
  • AI가 모든 일을 알아서 대신해주길 기대한다.
  • 본인의 문제 설명 없이 완성된 답만 원한다.
  • 이미 완성된 정답과 정형화된 레퍼런스만 보고 선택하고 싶다.

알림 구독 / 상담 신청

두 트랙 중 편한 방식으로 남겨주세요

첫 영상과 실험 노트를 가볍게 받아보고 싶다면 알림만 남기면 됩니다. 이미 자동화하고 싶은 업무가 있다면 베타 진단 세션 의도를 함께 적어주세요.

  • 실험 알림 받기 (무료, 1분)
  • 자동화 상담 신청 (베타, 무료): 30분 진단 희망
  • 자동화하고 싶은 반복 업무와 현재 해결 방식을 적어주세요.
  • 베타 진단 세션 희망 여부를 함께 남겨주세요.
